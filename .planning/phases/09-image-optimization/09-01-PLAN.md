<task type="auto">
  <name>Task 2: Create image audit script and validate zero broken URLs</name>
  <files>scripts/audit-images.js</files>
  <action>
Create `scripts/audit-images.js` that:

1. Scans ALL HTML files in `out/` directory recursively
2. Extracts every image URL from:
   - `src="..."` attributes (inline images)
   - `url(...)` in style attributes (background images)
   - `"image": "..."` in JSON-LD script tags (schema images)
3. Deduplicates URLs
4. For each unique URL:
   - If HTTP(S): sends a HEAD request, reports any non-200 status
   - If relative path: checks if file exists in `out/` directory
5. Checks for same-page duplicate images: for each HTML file, collects all image URLs (src + backgroundImage) and flags any URL appearing more than once on the same page
6. Reports:
   - Total images found
   - Broken URLs (with the page they appear on)
   - Same-page duplicates (URL + page)
   - R2 compliance: count of R2 URLs vs non-R2 URLs
   - Summary: PASS/FAIL

Use built-in Node.js only (no external dependencies). Use `fs`, `path`, and `https`/`http` modules. For HTTP HEAD requests, use native `https.request` with a concurrency limit of 10 to avoid overwhelming R2.

Exit code: 0 if zero broken URLs and zero non-R2 image URLs. Exit code 1 otherwise.

Run the script after build:
```bash
npm run build && node scripts/audit-images.js
```

If any broken URLs found, fix them in the source files and re-run until the script passes.
  </action>
