Create `scripts/audit-meta.js` that performs a comprehensive audit of all meta titles and descriptions. This is the Phase 3 quality gate script.

**The script must check ALL Phase 3 success criteria:**

1. **Title checks:**
   - Every page has a title containing "5 Star Roofing" (flag any missing)
   - Every title is under 60 characters (flag any over with char count)
   - No duplicate titles (use a Map to detect collisions; print both URLs for any duplicate)
   - No "TXService" typos

   - Every description is under 160 characters (flag any over with char count)
   - No descriptions start with lowercase letter
   - Every description has a CTA (contains "call", "Call", "free", "Free", "schedule", "Schedule", "(806)", or "622-6041")

   - No double commas
   - No duplicate descriptions (detect collisions)

   - Total pages scanned
   - Pass/fail for each check category
   - List of any failing pages with specific issues
   - Overall PASS/FAIL verdict

   - Flag any two pages that share the same primary keyword (potential cannibalization).

Run the script: `node scripts/audit-meta.js`
- Should output PASS for all checks
- If any FAIL, the issues must be fixed before proceeding
