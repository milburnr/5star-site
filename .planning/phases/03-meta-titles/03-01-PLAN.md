Create `scripts/fix-meta-titles.js` following the established pattern from `scripts/standardize-nap.js`. The script must:

1. **File discovery:** Recursively find all `page.tsx` files under `app/`.

- For pages missing brand entirely (52 pages): if title has multiple pipe segments like "Service | Free Inspection | Something", consolidate to "{Service} {City} TX | 5 Star Roofing". If single segment, append " | 5 Star Roofing".
- After all fixes, verify title is under 60 characters. If over, abbreviate the service name portion (not the brand name). Log any that can't be auto-shortened for manual review.
- For pages with openGraph.title (7 pages), update that field to match the new title.

**Modes:**
- `--dry-run`: Print all proposed changes without writing files. Show before/after for each title.
- `--report`: Output a summary table of all titles with character counts.
- No flag: Apply changes and write files.

**Output:** Print counts: total files scanned, titles changed, titles already correct, titles needing manual review (if any).
