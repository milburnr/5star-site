  <name>Task 1: Write and run AggregateRating removal script for all page.tsx files</name>
  <files>scripts/remove-aggregate-rating.js</files>
  <action>
Create a Node.js script at `scripts/remove-aggregate-rating.js` that:

1. Recursively finds all `page.tsx` files under `app/`
2. For each file, reads the content as a string
3. Removes ALL `aggregateRating` blocks using these regex patterns (order matters):

**Pattern 1 — aggregateRating followed by another property (remove block + trailing comma):**
```javascript
/"aggregateRating"\s*:\s*\{[^}]*\}\s*,\s*/g
```

**Pattern 2 — aggregateRating as last property before `}` (remove preceding comma + block):**
```javascript
/,\s*"aggregateRating"\s*:\s*\{[^}]*\}/g
```

**Pattern 3 — aggregateRating as ONLY property or first without preceding comma:**
```javascript
/"aggregateRating"\s*:\s*\{[^}]*\}\s*/g
```

Apply patterns in order: Pattern 1 first, then Pattern 2, then Pattern 3.

4. Writes the modified content back only if changes were made
5. Logs: total files scanned, files modified, files unchanged
6. Includes a `--dry-run` flag that shows what would change without writing

Run the script:
```bash
node scripts/remove-aggregate-rating.js --dry-run  # Preview first
node scripts/remove-aggregate-rating.js             # Execute
```

- The reviews page (`app/reviews/page.tsx`) should ALSO have AggregateRating removed (it claims 500 reviews but only shows ~12 testimonials -- still fabricated)

- Nested `{}` inside AggregateRating: verified by research that all AggregateRating objects are flat (no nested objects), so `[^}]*` is safe
